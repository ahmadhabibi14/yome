## Yome - Listen your fantastic audio

### How to start ?

Install dependencies
```pnpm install```

Start development
```pnpm dev```

Build to production
```pnpm build```

### Starter Pack ?

- Frontend library `React`
- Programming language `TypeScript`
- CSS framework `Tailwind`
- Build tool `Astro.build`
- Icons library `Iconsax`
- Package manager `pnpm`
